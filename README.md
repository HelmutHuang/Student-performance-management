# 学生表现管理系统 (Student Performance Management System)

一个基于纯前端技术（Native JavaScript）的学生日常表现、考勤和座位安排管理工具。本系统所有数据均存储在浏览器本地（LocalStorage），无需依赖任何后端服务即可独立运行和使用。

## ✨ 核心功能

* **📊 班级数据管理 (`3.班级数据管理.html`)**
    * **创建班级:** 支持输入班级名称，并通过文本框批量导入学生名单（支持从 Excel 或其他列表复制粘贴）。
    * **编辑班级:** 提供强大的编辑功能，支持修改班级名称、批量添加新学生到现有班级。
    * **学生排序:** 在编辑模式下，支持通过**拖拽**的方式调整学生在列表中的顺序。
    * **数据覆盖:** 支持同名班级的覆盖更新。

* **🪑 点名与座位安排 (`1.点名与座位.html`)**
    * **可视化座位布局:** 支持拖放操作来安排学生座位。
    * **自定义布局:** 可配置座位区域的行数、列数，并设置过道位置（支持多条过道）。
    * **智能点名:** 随机点名功能，自动排除已请假学生。点名结果弹窗中包含**“加分”**按钮，可直接对被点名学生进行奖励。
    * **考勤记录:** 通过拖放学生到不同的考勤类别（迟到、旷课、请假等）来实时记录表现。提供**垃圾桶**图标，拖拽学生至此可清除其当天的考勤状态。
    * **绿色标记:** 支持点击座位/列表中的学生进行临时标记（绿色），并通过点击考勤标题一键批量记录。
    * **数据导入/导出:** 支持将当前的座位布局和考勤状态导出为紧凑型代码，便于数据迁移或分享。

* **📈 出勤记录与成绩统计 (`2.出勤记录.html`)**
    * **历史记录管理:** 按日期查看所有已保存的考勤记录。支持**编辑**历史记录（修改日期、调整当天的考勤人员）和删除记录。
    * **学生汇总:** 点击学生姓名可查看其详细的考勤汇总次数。
    * **灵活评分:** 基础分 100 分，根据考勤记录自动扣分。
    * **额外加分:** 支持对单个学生进行自定义加分/减分操作（点击“修改”按钮）。
    * **最终成绩计算:** 采用独特的**满分群体平均分线性缩放**算法，将原始分数映射到 0-100 范围内，确保公平性。
    * **全部出勤表:** 一键生成包含所有学生考勤次数、统计分数和最终成绩的详细表格，并支持**一键复制**到剪贴板（兼容 Excel 粘贴）。

* **🧭 全局导航**
    * **浮动导航栏:** 页面顶部内置浮动导航栏，支持在不同功能模块间快速切换。

## �️ 技术栈

* **前端核心:** 原生 HTML5, CSS3, Native JavaScript (ES6+)
* **数据存储:** LocalStorage（用于持久化所有班级、学生、考勤和配置数据）
* **样式:** 简洁的 Flexbox/Grid 布局，使用 CSS 变量（`var(--primary-color)`）管理主题颜色。

## �🚀 快速开始

### 运行项目

1.  克隆或下载本项目文件。
2.  在任何现代浏览器中打开以下任意一个 HTML 文件即可启动应用：
    * `1.点名与座位.html`
    * `2.出勤记录.html`
    * `3.班级数据管理.html`

### 使用步骤

1.  **创建班级:** 
    * 导航至 `3.班级数据管理.html`。
    * 输入班级名称，并在文本框中粘贴多行学生姓名。
    * 点击 **“批量导入”** 和 **“添加班级”**。

2.  **安排座位和记录考勤:**
    * 导航至 `1.点名与座位.html`。
    * 选择你的班级。
    * 你可以拖动左侧的学生到右侧的座位上进行安排。
    * 通过拖动学生到下方的考勤区域（如“迟”、“旷”）来记录表现。

3.  **查看成绩:**
    * 导航至 `2.出勤记录.html`。
    * 选择班级，即可在左侧看到学生的统计分数和最终成绩。
